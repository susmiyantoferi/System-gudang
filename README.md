#
    Instalasi :
* Download or clone code
* Open code editor
* Edit file .env

        Example file .env connect mysql :
        DB_CONNECTION=mysql
        DB_HOST=127.0.0.1
        DB_PORT=3306
        DB_DATABASE=gudang
        DB_USERNAME=root
        DB_PASSWORD=


    Run application :
* Open code editor
* Open Terminal
* $ composer update
* $ php artisan migrate
* $ php artisan db:seed
* $ php artisan serve
